# Resume Website - HTML Assignment

## Overview
This is a single-page resume website built using only HTML.

## Features
- Header section with name and title
- About section
- Skills list
- Experience table
- Projects section
- Education table
- Contact details

## How to Run
1. Download or clone the repository
2. Open index.html in any browser
